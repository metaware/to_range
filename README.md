# ToRange

Small Ruby Gem to Convert strings (parseable) into Range Objects

## Installation

Add this line to your application's Gemfile:

    gem 'to_range'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install to_range

## Usage

    irb(main):001:0> "1..10".to_range
    => 1..10
    => PROFIT!
